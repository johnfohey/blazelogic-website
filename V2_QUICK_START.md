## Installation

### Option A: Fresh Install
1. Download `blazelogic-website-v2.zip`
2. Extract to your project folder
3. Push to GitHub
4. Enable GitHub Pages
5. Done!

### Option B: Upgrade from v1.0
1. Backup your current files
2. Replace:
   - `index.html`
   - `styles.css`
   - `script.js`
3. Add `VERSION_2_CHANGELOG.md` and `V2_QUICK_START.md`
4. Deploy
5. Done!

**Migration time: < 5 minutes, no breaking changes**
