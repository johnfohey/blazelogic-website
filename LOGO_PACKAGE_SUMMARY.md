## Package Contents

### SVG Vector Files (6 files)

All SVG files are resolution-independent, scalable, and optimized for web and digital use.

#### 1. `logo-primary-horizontal.svg`
- **Purpose:** Full logo with wordmark, primary use
- **Dimensions:** 800x200px artboard
- **Best for:** Website headers, marketing materials, digital displays
- **Contains:** 
  - Metallic "B" icon
  - Circuit traces and nodes
  - "BLAZE" + "LOGIC" wordmark
  - Blue pulse underline
- **Colors:** Full color (blue, orange, white, gray)
- **Background:** Dark (#0D1117)

#### 2. `logo-icon-only.svg`
- **Purpose:** Icon-only version for small applications
- **Dimensions:** 200x200px artboard (square)
- **Best for:** Favicon, app icons, avatars, social media profiles
- **Contains:**
  - Metallic "B" with circuit traces
  - Orange accent nodes
  - No wordmark
- **Colors:** Full color
- **Background:** Dark (#0D1117)

#### 3. `logo-app-icon-square.svg`
- **Purpose:** Official app icon with rounded corners
- **Dimensions:** 512x512px artboard (square with rounded corners)
- **Best for:** iOS App Store, Android Play Store, web manifest, app launchers
- **Contains:**
  - Large metallic "B"
  - Circuit traces (scaled up)
  - Orange accent nodes
  - Gradient depth effect
- **Colors:** Full color
- **Background:** Dark with gradient
- **Special:** Rounded corners (80px radius) built into design

#### 4. `logo-monochrome-light.svg`
- **Purpose:** Dark version for light backgrounds
- **Dimensions:** 800x200px artboard (same as primary)
- **Best for:** Light background websites, printed materials on light stock, business cards
- **Contains:**
  - Dark "B" (#0D1117)
  - Dark circuit traces and wordmark
  - Inverse of primary design
- **Colors:** Single color (dark #0D1117)
- **Background:** Light (white, cream, light gray)

#### 5. `logo-monochrome-dark.svg`
- **Purpose:** Light version for dark backgrounds
- **Dimensions:** 800x200px artboard (same as primary)
- **Best for:** Dark background websites, presentations, dark mode UI, inverse cases
- **Contains:**
  - Light "B" (#E5E7EB)
  - Light circuit traces and wordmark
- **Colors:** Single color (light #E5E7EB)
- **Background:** Dark (dark gray, black, navy)

#### 6. `logo-stacked-vertical.svg`
- **Purpose:** Vertical arrangement of logo and wordmark
- **Dimensions:** 400x600px artboard (2:3 aspect ratio)
- **Best for:** Narrow vertical spaces, book spines, portrait layouts, social media stories
- **Contains:**
  - Icon on top
  - Wordmark below
  - Full color design
- **Colors:** Full color (blue, orange, white, gray)
- **Background:** Dark (#0D1117)

## Key Specifications

### Colors

| Element | Color Name | Hex Code | RGB | Usage |
|---------|-----------|----------|-----|-------|
| Dark Background | BlazeLogic Dark | #0D1117 | 13, 17, 23 | Primary dark |
| Light Text | BlazeLogic Light | #E5E7EB | 229, 231, 235 | Text, light elements |
| Technology Blue | Blue | #2563EB | 37, 99, 235 | Circuit traces, underline |
| Accent Orange | Orange | #FF6A00 | 255, 106, 0 | "LOGIC" text, nodes |

### Minimum Sizes

| Use Case | Minimum Size |
|----------|--------------|
| Web icon | 32px |
| App icon | 60px |
| Print icon | 1 inch (254px @ 254 DPI) |
| Logo on web | 320px wide |

### Clear Space

- Horizontal logo: 1/4 logo height on left/right, 1/8 on top/bottom
- Icon logo: 1/6 of width/height on all sides

## Recommended Implementations

### For Websites

1. **Header/Hero:**
   - File: `logo-primary-horizontal.svg`
   - Size: 400-800px responsive
   - Format: SVG inline or img tag

2. **Navbar:**
   - File: `logo-icon-only.svg`
   - Size: 48-64px
   - Format: SVG img tag

3. **Favicon:**
   - File: `logo-icon-only.svg`
   - All sizes from 32px to 512px
   - Format: SVG + PNG fallback

### For Mobile Apps

1. **App Icon:**
   - File: `logo-app-icon-square.svg` or PNG
   - Sizes: 180px (iOS), 192px (Android), 512px (web)
   - Format: PNG for app stores

2. **In-app Logo:**
   - File: `logo-icon-only.svg`
   - Size: 40-60px
   - Format: SVG

### For Print

1. **Business Card:**
   - File: `logo-primary-horizontal.svg` or PNG-300DPI
   - Size: 1"x0.25" to 2"x0.5"
   - Format: PDF with embedded fonts

2. **Poster:**
   - File: `logo-primary-horizontal-print-3072.png`
   - Resolution: 300 DPI
   - Size: 2" to 10" wide

- Monochrome dark on dark: 13.1:1 (exceeds WCAG AAA)
